Registration Management Guide

This panel allows you to manage user registration applications from the Create Account page.

  • Pending Registrations: Review and process new applications
  • Approved Accounts: View successfully created user accounts
  • Rejected Applications: Review declined registrations
  • Create Manual: Manually create user accounts for special cases
  • All Registrations: View complete registration history

Note: All changes require manual save. Remember to click "Save All" after making changes.
